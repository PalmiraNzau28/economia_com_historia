# Backend

Base inicial do backend do projecto.

## Estrutura

- `src/server.ts` - ponto de entrada do servidor
- `src/app.ts` - configuracao da app Express
- `src/routes` - rotas da API
- `src/controllers` - controladores
- `src/middlewares` - middlewares globais
- `src/config` - variaveis de ambiente e configuracoes
